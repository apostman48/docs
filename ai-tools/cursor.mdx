# JSON Viewer Demo

This page demonstrates the `JsonViewer` snippet with multiple data sources.

## 2) JSON string via `json` (fully expanded)

<JsonViewer json={sampleJson} collapsedAt={0} />

---

## 3) Large array (deterministic values)

## 4) Invalid JSON handling
