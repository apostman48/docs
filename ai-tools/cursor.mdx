# JSON Viewer Demo

This page demonstrates the `JsonViewer` snippet with objects, JSON strings, arrays, and invalid JSON.

---

## 1) Object via `data`

<JsonViewer
  collapsedAt={1}
  data={{
    meta: { requestId: "req_9c1e2c0f", receivedAt: "2025-09-07T10:31:00Z" },
    user: {
      id: 123456,
      name: { first: "Ada", last: "Lovelace" },
      roles: ["admin", "analyst"],
      profile: { bio: "Mathematician", lastLoginAt: null }
    },
    projects: [
      { id: "prj_1", name: "Analytical Engine", stats: { issuesOpen: 12 } },
      { id: "prj_2", name: "Notes on Babbage", stats: { issuesOpen: 0 } }
    ]
  }}
/>

---

## 2) JSON string via `json` (fully expanded)

<JsonViewer
  collapsedAt={0}
  json={`{
    "status": "ok",
    "version": "2.1.0",
    "endpoints": [
      {"path": "/v2/users", "method": "GET"},
      {"path": "/v2/projects/:id", "method": "PATCH"}
    ]
  }`}
/>

---

## 3) Large array (deterministic values)

<JsonViewer
  collapsedAt={1}
  data={{
    results: Array.from({ length: 10 }).map((_, i) => ({
      id: i + 1,
      ok: (i % 3) !== 0,
      score: ((i * 137) % 1000) / 10
    })),
    paging: { page: 1, pageSize: 10, total: 5000 }
  }}
/>

---

## 4) Invalid JSON handling

<JsonViewer json={`{ "oops": true, trailing: }`} />
