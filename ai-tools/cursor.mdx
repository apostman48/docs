## How to Use

1. **Create the snippet**: Save the first file as `json-counter.mdx` in your `snippets` folder

2. **Use in your docs**: Add the JsonItemCounter component to any MDX page:

```jsx
<JsonItemCounter 
  title="Your Custom Title"
  data={yourJsonObject}
/>
```
